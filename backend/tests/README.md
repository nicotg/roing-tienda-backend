# Backend tests

Coloca aquí los tests unitarios del backend (TypeScript).

Convenciones sugeridas:
- Archivos: `*.test.ts`
- Organización: puedes colocar los tests aquí o en `src/**/__tests__` cerca del código a probar.
- Cada test debe ser independiente y no depender de estado externo.

Próximos pasos:
1. Instalar Jest + ts-jest + @types/jest
2. Agregar scripts de `npm test`
3. Configurar `jest.config.ts`
4. Escribir el primer test (p. ej. para `src/helpers/generate-jwt-helper.ts`)
